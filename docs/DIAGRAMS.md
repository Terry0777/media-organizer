## 2. 数据流图

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐

└──────────┘     └──────────┘     └──────────┘     └──────────┘     └──────────┘

                                           ▼                ▼                ▼

                                   │              SQLite Database              │
```
