**版本：** v1.0  
**创建日期：** 2026-03-04  
**设计师：** 虾米 🦐

## 1. 设计原则

### 1.1 核心理念

```
📸 内容优先 - 媒体文件是主角，UI 要隐身
⚡ 性能至上 - 万级文件也要流畅
🎯 操作直观 - 减少学习成本
🌙 舒适护眼 - 深色模式必备
```

### 1.2 设计灵感

| 参考产品 | 借鉴点 |
|----------|--------|
| Google Photos | 时间线、智能分组 |
| Adobe Bridge | 专业元数据显示 |
| Eagle | 标签管理、收藏系统 |
| digiKam | 本地优先、功能全面 |
| macOS Photos | 流畅动画、简洁交互 |

## 8. 动画与过渡

### 8.1 微交互

| 场景 | 动画 | 时长 |
|------|------|------|
| 悬停卡片 | 轻微放大 (1.05x) | 150ms |
| 选中卡片 | 边框高亮 + 勾选 | 100ms |
| 切换视图 | 淡入淡出 | 200ms |
| 加载缩略图 | 渐显 | 300ms |
| 删除文件 | 缩小 + 消失 | 250ms |
| 添加标签 | 标签芯片滑入 | 200ms |

### 8.2 页面过渡

```
时间线 → 标签视图:

打开详情面板:
[面板从右侧滑入]

关闭应用:
[窗口缩小到任务栏]
```

## 9. 无障碍设计

### 9.1 键盘导航

- Tab 键遍历所有可交互元素
- Enter 确认，Esc 取消
- 方向键浏览媒体网格
- Space 选择/取消选择

### 9.2 屏幕阅读器

- 所有图标有 aria-label
- 图片缩略图有 alt 描述
- 动态内容有 live region

### 9.3 对比度

- 文字对比度 ≥ 4.5:1 (WCAG AA)
- 大文字对比度 ≥ 3:1

- [Lucide Icons](https://lucide.dev/) - 开源，风格统一
- [Heroicons](https://heroicons.com/) - Tailwind 官方
- [Remix Icon](https://remixicon.com/) - 中文友好

### 10.2 UI 组件库

- [shadcn/ui](https://ui.shadcn.com/) - 推荐，高度可定制
- [Radix UI](https://www.radix-ui.com/) - 无障碍优先
- [Headless UI](https://headlessui.com/) - Tailwind 搭档

### 10.3 设计工具

- Figma - UI 设计
- Excalidraw - 草图绘制
- Storybook - 组件文档
