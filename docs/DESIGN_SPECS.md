## 3. 字体系统

### 3.1 字体家族

```
主字体：Inter (推荐) 或 System UI
备用：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial

代码字体：JetBrains Mono 或 SF Mono
中文：PingFang SC (macOS) / Microsoft YaHei (Windows)
```

### 3.2 字号规范

| 用途 | 字号 | 字重 | 行高 | 字间距 |
|------|------|------|------|--------|
| 大标题 | 32px | Bold (700) | 40px | -0.5px |
| 标题 1 | 24px | SemiBold (600) | 32px | -0.3px |
| 标题 2 | 20px | SemiBold (600) | 28px | -0.2px |
| 标题 3 | 17px | Medium (500) | 24px | -0.1px |
| 正文 | 14px | Regular (400) | 20px | 0 |
| 辅助文字 | 12px | Regular (400) | 16px | 0 |
| 微小文字 | 11px | Regular (400) | 14px | 0 |

## 4. 间距系统

### 4.1 8px 基线网格

```
间距等级：4, 8, 12, 16, 20, 24, 32, 40, 48, 64, 80, 96

常用间距:
├── xs:  4px   (极小间距)
├── sm:  8px   (小组件间距)
├── md:  16px  (标准间距)
├── lg:  24px  (大间距)
├── xl:  32px  (区块间距)
└── 2xl: 48px  (大区块间距)
```

```
按钮：
├── Small:   4px 12px

├── Default: 16px
└── Large:   24px
```

## 6. 阴影系统

### 6.1 浅色模式阴影

```
Shadow / Sm
├── x: 0, y: 1, blur: 2, spread: 0
├── color: rgba(0, 0, 0, 0.05)
└── 应用：小按钮、标签芯片

Shadow / Md
├── x: 0, y: 4, blur: 8, spread: 0
├── color: rgba(0, 0, 0, 0.1)
└── 应用：卡片、下拉菜单

Shadow / Lg
├── x: 0, y: 8, blur: 24, spread: 0
├── color: rgba(0, 0, 0, 0.15)
└── 应用：模态框、浮动面板

Shadow / Xl
├── x: 0, y: 16, blur: 48, spread: 0
├── color: rgba(0, 0, 0, 0.2)
└── 应用：重要提示、拖拽中
```

### 6.2 深色模式阴影

```
深色模式使用更淡的阴影 + 边框组合
Shadow color: rgba(0, 0, 0, 0.3)
同时添加 1px border: #48484A
```

## 10. 设计检查清单

### 10.1 一致性检查

- [ ] 所有颜色使用 Design Token
- [ ] 所有字号使用 Typography Scale
- [ ] 所有间距使用 8px 基线
- [ ] 所有圆角使用统一等级
- [ ] 所有阴影使用预定义值

### 10.2 可用性检查

- [ ] 文字对比度 ≥ 4.5:1
- [ ] 点击区域 ≥ 44x44px
- [ ] 表单有明确标签
- [ ] 错误状态清晰可见
- [ ] 加载状态有反馈

### 10.3 响应式检查

- [ ] Desktop (1440px) 布局完整
- [ ] Laptop (1024px) 侧边栏可折叠
- [ ] Tablet (768px) 单栏布局
- [ ] Mobile (375px) 触摸友好
